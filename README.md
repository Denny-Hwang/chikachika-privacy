# ChikaChika Buddy - Privacy Policy

This repository hosts the privacy policy page for **ChikaChika Buddy**, a fun toothbrushing companion app for kids.

## Overview

The privacy policy is served as a static HTML page (`index.html`) via GitHub Pages.

## Key Points

- ChikaChika Buddy does **not** collect, store, or transmit any personal data
- All app data is stored locally on the user's device
- No third-party analytics, ads, or tracking services are used
- The app includes a Mirror Mode with AR effects — all face data is processed on-device and never stored
